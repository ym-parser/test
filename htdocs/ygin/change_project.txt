20130408
Для внешней части сайта теперь используется прикладной бутстрап, который находится в папке /protected/assets/ и может быть легко заменён и отредактирован для нужд сайта.
Вместо подключения файла в шаблоне
  $this->registerJsFile('bootstrap.js', 'ygin.assets.bootstrap.js');
  $this->registerCssFile('bootstrap.min.css', 'webroot.themes.business.bootstrap.css');
Стоит использовать подключение из пакета:
  Yii::app()->clientScript->registerCoreScript('bootstrap'); 

Данный файл - webroot.themes.business.bootstrap.css - стоит на всякий случай перенести в /protected/assets/bootstrap/css, т.к. он может содержать проектные кастомизации


20130420
В связи с переходом на новый алгоритм хеширования паролей была изменена модель User:
удалены методы User::hashPassword, User::getIsPasswordChanged, поэтому если вы использовали их,
то эти куски кода надо переписать.

Перевод сайтов на Yii (старых) на новый алгоритм хеширования:
1) Скопировать с engine_macro папку ygin/modules/user/behaviors/password
2) Добавить в таблицу da_users следующие поля:
ALTER TABLE `da_users` ADD `requires_new_password` TINYINT(1) COMMENT  'Необходимо сменить пароль';
ALTER TABLE  `da_users` ADD  `salt` VARCHAR( 255 ) NULL COMMENT  'Соль для пароля',
    ADD  `password_strategy` VARCHAR( 255 ) NULL COMMENT  'Стратегия для формирования пароля';
3) прогнать запрос UPDATE `da_users` SET `password_strategy` = 'legacy';
4) подключить в модели User поведения

  public function behaviors() {
    return array(
      "APasswordBehavior" => array(
        "class" => "APasswordBehavior",
        "defaultStrategyName" => "bcrypt",
        "passwordAttribute" => "user_password",
        "strategies" => array(
          "bcrypt" => array(
            "class" => "DaBcryptPasswordStrategy",
            "workFactor" => 14
          ),
          "legacy" => array(
            "class" => "ALegacyMd5PasswordStrategy",
          )
        ),
      ),
    );
  }
5) убрать использование методов  User::hashPassword, User::getIsPasswordChanged,
 и формирования хеша пароля напрямую, т.е. $user->user_password = md5($password);
 Нужно использовать только $user->save(), поведения сделают за вас всю работу.
