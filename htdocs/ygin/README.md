ygin
====

Система ygin - бесплатная open-source CMF-система, в основе которой лежит фреймворк [yii](http://www.yiiframework.com/).

Система позволяет быстро разворачивать проекты средней сложности, легко создавать новые сущности, управлять ими с помощью готовых удобных инструментов.
Система ygin является каркасом, базой, бэкендом для построения прикладной информационной системы. 
Основная цель - дать возможность разработчикам писать больше прикладной код и тратить меньше времени на организацию и администрирование данных. 
Всё что можно сделать общим, пусть будет сделано и доступно всем.

Данную систему можно применить в качестве системы управления сайтом (CMS).
В настоящее время система содержит пару десятков готовых модулей, позволяющих быстро создать сайт, содержащий уже привычные функциональные модули:  
-интернет-магазин  
-опросы  
-баннеры  
-викторины  
-новости  
-комментарии  
-вопрос-ответ  
-формы обратной связи  
-фотогалерея  
-поиск  
На базе системы достаточно легко написать свои решения или изменить существующие под свою прикладную логику.

Подробнее с возможностями и документацией системы можно ознакомиться на [официальном сайте](http://ygin.ru).

Требования к системе
--------------------

Минимальные требования для работы системы аналогичны требованиям фреймворка yii.
Единственное дополнительное требование - наличие СУБД MySQL, 5+
